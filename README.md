# Employee Management System

## Description
Employee Management System developed using Java Swing, JDBC, and MySQL.
This desktop application allows users to manage employee records with CRUD operations.

## Technologies Used
- Java (Core Java)
- Java Swing (GUI)
- JDBC
- MySQL
- Eclipse IDE

## Features
- Add Employee
- View Employee
- Update Employee
- Delete Employee
- MySQL Database Connectivity

## Database
- MySQL
- Tables: employee
